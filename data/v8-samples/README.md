## 🚀 快速运行

```bash
# 进入目录
cd stock_selection_v8_1_standalone

# 1) 主选股 (生成 Excel, 约 30 秒)
python run.py

# 2) T+1 预估验证 (生成 独立验证Excel, 约 10 秒)
python verify.py

# 3) 合并验证结果到主Excel
python merge.py

# 完成后, output/ 下就是最终的8-Sheet主Excel
```
