# 向通达信公式设置数据 formula_set_data

在调用公式前须先设置公式参数，此接口与 `formula_set_data_info` 作用一样，会互相覆盖。

> 需要先下载对应的盘后数据。设置的数据在断开连接前一直生效，后设置的数据会覆盖前面设置的数据。

## 函数签名

```python
formula_set_data(stock_code: str = '',
                 stock_period: str = '1d',
                 stock_data: List = [],
                 count: int = 1,
                 dividend_type: int = 0)
```

## 输入参数

| 参数 | 是否必选 | 参数类型 | 参数说明 |
|------|----------|----------|----------|
| `stock_code` | Y | `str` | 股票代码 |
| `stock_period` | Y | `str` | K 线周期 |
| `stock_data` | Y | `List` | 指定格式的 K 线数据 |
| `count` | Y | `int` | 选取的 K 线数量 |
| `dividend_type` | Y | `int` | 复权类型 |

### dividend_type 说明

| 值 | 说明 |
|----|------|
| `0` | 不复权 |
| `1` | 前复权 |
| `2` | 后复权 |

### count 说明

- `count` 为设定 `stock_data` 中生效的 K 线数据，即 `stock_data` 中有效数据不能小于 `count`。
- `count` 须大于 0，且最大不超过 24000。

## 接口使用

```python
from tqcenter import tq

tq.initialize(__file__)

test_md = tq.get_market_data(stock_list=['688318.SH'], count=5, period='1d')
format_md = tq.formula_format_data(test_md)
formula_set_k = tq.formula_set_data(
    stock_code='688318.SH',
    stock_period='1d',
    stock_data=format_md['688318.SH'],
    count=len(format_md['688318.SH']))
print(formula_set_k)
```
