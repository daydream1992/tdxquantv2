# 获取指定种类的公式列表 formula_get_all

获取指定种类的公式列表。

## 函数签名

```python
def formula_get_all(formula_type: int = 0)
```

## 输入参数

| 参数 | 是否必选 | 参数类型 | 参数说明 |
|------|----------|----------|----------|
| `formula_type` | 是 | `int` | 公式种类标识（见下表） |

### formula_type 取值

| 值 | 说明 |
|----|------|
| `0` | 技术指标公式 |
| `1` | 条件选股公式 |
| `2` | 专家系统公式 |

## 输出数据

| 名称 | 类型 | 说明 |
|------|------|------|
| `acCode` | `str` | 公式代码 |
| `acName` | `str` | 公式名称 |
| `isSys` | `int` | 是否为系统公式 |

## 接口使用

```python
from tqcenter import tq

tq.initialize(__file__)

formule_all = tq.formula_get_all(formula_type=0)
print(formule_all)
```
