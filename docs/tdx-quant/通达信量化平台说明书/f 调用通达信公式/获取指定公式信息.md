# 获取指定公式信息 formula_get_info

获取指定公式信息。

## 函数签名

```python
def formula_get_info(formula_type: int = 0, formula_code: str = '')
```

## 输入参数

| 参数 | 是否必选 | 参数类型 | 参数说明 |
|------|----------|----------|----------|
| `formula_type` | 是 | `int` | 公式种类标识 |
| `formula_code` | 是 | `str` | 公式代码 |

### formula_type 取值

| 值 | 说明 |
|----|------|
| `0` | 技术指标公式 |
| `1` | 条件选股公式 |
| `2` | 专家系统公式 |

## 输出数据

| 名称 | 类型 | 说明 |
|------|------|------|
| `acCode` | `str` | 公式代码 |
| `acName` | `str` | 公式名称 |
| `isSys` | `int` | 是否为系统公式 |
| `ParaNum` | `int` | 入参数量 |
| `Para` | `Set` | 公式入参参数（含 `ParaName`、`Min`、`Max`、`Default`） |
| `LineNum` | `int` | 出参数量 |
| `Line` | `Set` | 公式出参参数（含 `LineName`） |

## 接口使用

```python
from tqcenter import tq

tq.initialize(__file__)

formule_all = tq.formula_get_all(formula_type=0)
print(formule_all)
```
