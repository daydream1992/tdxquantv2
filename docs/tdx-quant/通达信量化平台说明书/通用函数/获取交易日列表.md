# 获取交易日列表 get_trading_dates

根据指定时间段获取交易日列表。

## 函数签名

```python
get_trading_dates(market: str,
                  start_time: str,
                  end_time: str,
                  count: int = -1) -> List
```

## 输入参数

| 参数 | 是否必选 | 参数类型 | 参数说明 |
|------|----------|----------|----------|
| `market` | 是 | `str` | 市场代码（暂固定为 `SH`） |
| `start_time` | 否 | `str` | 起始日期 |
| `end_time` | 否 | `str` | 结束日期 |
| `count` | 否 | `int` | 返回最近的 count 个交易日 |

## 参数说明

- 需要先在客户端下载上证指数（999999）的盘后数据，目前仅支持 A 股。
- `count > 0` 时，限制返回从结束日期往前最近的 count 个在限定时间段中的交易日。

## 接口使用

```python
from tqcenter import tq

tq.initialize(__file__)

trade_dates = tq.get_trading_dates(market='SH', start_time='20220101', end_time='', count=10)
print(trade_dates)
```

## 数据样本

```python
['20251211', '20251212', '20251215', '20251216', '20251217', '20251218', '20251219', '20251222', '20251223', '20251224']
```
