# 刷新行情缓存

## 刷新行情缓存 refresh_cache

刷新行情缓存（最新 snapshot 和 K 线数据）。如果不调用，首次取 snapshot 和 K 线时系统会自动刷新一次行情。

### 函数签名

```python
def refresh_cache(market: str = 'AG',
                  force: bool = False)
```

### 输入参数

| 参数 | 是否必选 | 参数类型 | 参数说明 |
|------|----------|----------|----------|
| `force` | 是 | `bool` | 是否强制刷新 |
| `market` | 是 | `str` | 指定刷新的市场 |

### 参数说明

- `force` 为 `False` 时距离上次刷新不足 10 分钟则不会刷新，为 `True` 时强制刷新。
- `market` 赋值：

| 值 | 说明 |
|----|------|
| `AG` | A股 |
| `HK` | 港股 |
| `US` | 美股 |
| `QH` | 国内期货 |
| `QQ` | 股票期权 |
| `NQ` | 新三板 |
| `ZZ` | 中证和国证等指数 |
| `OF` | 基金净值 |
| `ZS` | 沪深京指数 |
| `OJ` | 期货期权 |

### 接口使用

```python
from tqcenter import tq

tq.initialize(__file__)

refresh_cache = tq.refresh_cache()
print(refresh_cache)
```

### 数据样本

使用后会在客户端弹出刷新数据的加载界面，加载完成后才会有返回。

```json
{
   "Error": "Refresh Cache Success.",
   "ErrorId": "0",
   "run_id": "1"
}
```

## 刷新历史K线缓存 refresh_kline

根据股票和周期刷新历史 K 线缓存，如果本地没有下载完整的日线等数据，则可以调用这个函数定向下载某些品种某些周期的历史 K 线数据。

### 函数签名

```python
refresh_kline(stock_list: List[str] = [], period: str = '')
```

### 输入参数

| 参数 | 是否必选 | 参数类型 | 参数说明 |
|------|----------|----------|----------|
| `stock_list` | 是 | `List[str]` | 证券代码列表，格式为 6 位数+市场后缀（.SH/.SZ/.BJ 等） |
| `period` | 是 | `str` | 周期：`1d` 日线、`1m` 一分钟线、`5m` 五分钟线，只支持这三种，其它周期的数据均由这三种数据生成 |

### 接口使用

```python
from tqcenter import tq

tq.initialize(__file__)

refresh_kline = tq.refresh_kline(stock_list=['688318.SH'], period='1d')
print(refresh_kline)
```

### 数据样本

> 注：如果在盘中交易时间段下载 1m 和 5m 分钟线，只能下载到截止上个交易日的数据。

使用后会在客户端弹出刷新数据的加载界面，加载完成后才会有返回。

```json
{
   "Error": "refresh kline cache success.",
   "ErrorId": "0",
   "run_id": "1"
}
```
