# 获取指定日期市场交易数据 get_scjy_value_by_date

获取指定时间的市场交易数据。需要先在客户端中下载股票数据包。

## 函数签名

```python
get_scjy_value_by_date(field_list: List[str] = [],
                       year: int = 0,
                       mmdd: int = 0) -> Dict
```

## 输入参数

| 参数 | 是否必选 | 参数类型 | 参数说明 |
|------|----------|----------|----------|
| `field_list` | 是 | `List[str]` | 字段筛选，不能为空 |
| `year` | 是 | `int` | 指定年份 |
| `mmdd` | 是 | `int` | 指定月日 |

### 参数说明

- `year=0, mmdd=0`：取最新数据
- `year=0, mmdd=1`：取倒数第 2 个数据
- `year=0, mmdd=2`：取倒数第 3 个数据，依此类推

> 需要先在客户端中下载股票数据包。

## 输出数据

同 `get_scjy_value` 一样，返回 `Dict`，key 为字段名，value 为对应的值列表。

## 接口使用

```python
from tqcenter import tq

tq.initialize(__file__)

sc_one = tq.get_scjy_value_by_date(
    field_list=['SC6', 'SC7', 'SC8', 'SC9', 'SC10'],
    year=0,
    mmdd=0)
print(sc_one)
```
