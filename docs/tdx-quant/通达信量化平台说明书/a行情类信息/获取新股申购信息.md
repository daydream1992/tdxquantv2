# 获取新股申购信息 get_ipo_info

获取今天及未来的新股或新发债申购信息。

## 函数签名

```python
get_ipo_info(ipo_type: int = 0,
             ipo_date: int = 0)
```

## 输入参数

| 参数 | 是否必选 | 参数类型 | 参数说明 |
|------|----------|----------|----------|
| `ipo_type` | Y | `int` | 信息类型：`0` 新股申购信息、`1` 新发债信息、`2` 新股和新发债信息 |
| `ipo_date` | Y | `int` | 日期范围：`0` 只获取今天信息、`1` 获取今天及以后信息 |

## 返回数据

| 数据 | 默认返回 | 数据类型 | 数据说明 |
|------|----------|----------|----------|
| `Code` | Y | `str` | 证券代码 |
| `Name` | Y | `str` | 证券名称 |
| `SGDate` | Y | `str` | 申购日期 |
| `SGPrice` | Y | `str` | 申购价格 |
| `SGCode` | Y | `str` | 申购代码 |
| `MaxSG` | Y | `str` | 申购上限 |
| `PE_Issue` | Y | `str` | 发行市盈率 |

## 接口使用

```python
from tqcenter import tq

tq.initialize(__file__)

ipo_info = tq.get_ipo_info(ipo_type=2, ipo_date=1)
print(ipo_info)
```
