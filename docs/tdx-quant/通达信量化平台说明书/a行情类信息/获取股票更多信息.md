# 获取股票更多信息 get_more_info

获取指定股票更细节的信息。

## 函数签名

```python
def get_more_info(stock_code: str = '',
                  field_list: List = [])
```

## 输入参数

| 参数 | 是否必选 | 参数类型 | 参数说明 |
|------|----------|----------|----------|
| `stock_code` | 是 | `str` | 股票代码 |
| `field_list` | 否 | `List[str]` | 字段筛选，传空则返回全部 |

## 返回数据

### 基本与形态

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `MainBusiness` | 是 | `str` | 主营构成 |
| `SafeValue` | 是 | `str` | 安全分 |
| `ShineValue` | 是 | `str` | 亮点数 |
| `ShapeValue` | 是 | `str` | 短期形态+中期形态+长期形态编号 |
| `TPFlag` | 是 | `str` | 停牌标识 |
| `ZTPrice` | 是 | `str` | 涨停价 |
| `DTPrice` | 是 | `str` | 跌停价 |
| `HqDate` | 是 | `str` | 行情日期 |

### 成交量与市值

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `fHSL` | 是 | `str` | 换手率 |
| `fLianB` | 是 | `str` | 量比 |
| `Wtb` | 是 | `str` | 委比 |
| `Zsz` | 是 | `str` | 总市值（亿） |
| `Ltsz` | 是 | `str` | 流通市值（亿） |
| `vzangsu` | 是 | `str` | 量涨速 |
| `Fzhsl` | 是 | `str` | 分钟换手率 |
| `FzAmo` | 是 | `str` | 2 分钟金额（万元） |
| `FreeLtgb` | 是 | `str` | 自由流通股本（万） |

### 涨幅类

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `VOpenZAF` | 是 | `str` | 抢筹涨幅 |
| `ZAF` | 是 | `str` | 涨幅 |
| `ZAFYesterday` | 是 | `str` | 昨日涨幅 |
| `ZAFPre2D` | 是 | `str` | 前天涨幅 |
| `ZAFPre5` | 是 | `str` | 5 日涨幅 |
| `ZAFPre10` | 是 | `str` | 10 日涨幅 |
| `ZAFPre20` | 是 | `str` | 20 日涨幅 |
| `ZAFPre30` | 是 | `str` | 30 日涨幅 |
| `ZAFPre60` | 是 | `str` | 60 日涨幅 |
| `ZAFYear` | 是 | `str` | 年初至今涨幅 |
| `ZAFPreMyMonth` | 是 | `str` | 涨幅（本月来） |
| `ZAFPreOneYear` | 是 | `str` | 涨幅（一年来） |
| `ConZAFDateNum` | 是 | `str` | 连涨天数 |

### 资金流向

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `Zjl` | 是 | `str` | 主买净额（万元） |
| `Zjl_HB` | 是 | `str` | 主力净流入（万元） |
| `TotalBVol` | 是 | `str` | 总买量 |
| `TotalSVol` | 是 | `str` | 总卖量 |
| `BCancel` | 是 | `str` | 总撤买量 |
| `SCancel` | 是 | `str` | 总撤卖量 |
| `L2TicNum` | 是 | `str` | L2 逐笔成交数 |
| `L2OrderNum` | 是 | `str` | L2 逐笔委托数 |

### 涨停封板

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `FCAmo` | 是 | `str` | 封单额（万元） |
| `FCb` | 是 | `str` | 封成比 |
| `OpenAmo` | 是 | `str` | 开盘金额（万元）（A 股和板块指数有效） |
| `OpenZTBuy` | 是 | `str` | 竞价涨停买入金额（万元） |
| `OpenAmoPre1` | 是 | `str` | 昨开盘金额（万元） |
| `OpenVolPre1` | 是 | `str` | 昨开盘量 |
| `CJJEPre1` | 是 | `str` | 昨成交额（万元） |
| `CJJEPre3` | 是 | `str` | 3 日成交额（万元） |
| `FDEPre1` | 是 | `str` | 昨封单额（万元） |
| `FDEPre2` | 是 | `str` | 前封单额（万元） |
| `ZTGPNum` | 是 | `str` | 板块指数的涨停家数 |
| `LastStartZT` | 是 | `str` | 几天 |
| `LastZTHzNum` | 是 | `str` | 几板 |
| `EverZTCount` | 是 | `str` | 连板天 |
| `YearZTDay` | 是 | `str` | 年涨停天数 |

### 价格与估值

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `MA5Value` | 是 | `str` | 5 日均价 |
| `HisHigh` | 是 | `str` | 52 周最高 |
| `HisLow` | 是 | `str` | 52 周最低 |
| `IPO_Price` | 是 | `str` | 发行价 |
| `More_YJL` | 是 | `str` | ETF, LOF 溢价率 |
| `BetaValue` | 是 | `str` | 贝塔系数 |
| `DynaPE` | 是 | `str` | 动态市盈率 |
| `MorePE` | 是 | `str` | 市盈率（港股：动，其他扩展：静） |
| `StaticPE_TTM` | 是 | `str` | 市盈率（TTM） |
| `DYRatio` | 是 | `str` | 股息率 |
| `PB_MRQ` | 是 | `str` | 市净率（MRQ） |

### 类型标识

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `IsT0Fund` | 是 | `str` | 是否是 T+0 基金 |
| `IsZCZGP` | 是 | `str` | 是否是注册制 A 股 |
| `IsKzz` | 是 | `str` | 是否是可转债 |
| `Kzz_HSCode` | 是 | `str` | 可转债对应的正股代码 |
| `QHMainYYMM` | 是 | `str` | 主力合约关联的月份（期货），主力和次主力 |
| `Yield` | 是 | `str` | 应计利息（债券），占款天数（回购） |

### 财务指标

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `KfEarnMoney` | 是 | `str` | 扣非净利润（万元） |
| `RDInputFee` | 是 | `str` | 研发费用（万元） |
| `CashZJ` | 是 | `str` | 货币资金（万元） |
| `PreReceiveZJ` | 是 | `str` | 合同负债（万元） |
| `OtherQYJzc` | 是 | `str` | 其它权益工具（万元） |
| `StaffNum` | 是 | `str` | 员工人数 |

### 关键日期

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `RecentGGJYDate` | 是 | `str` | 最近北上大额交易日 |
| `RecentHGDate` | 是 | `str` | 最近回购预案日 |
| `RecentIncentDate` | 是 | `str` | 最近股权激励预案日 |
| `NoticeDate_Recent` | 是 | `str` | 最近业绩预告日 |
| `RecentReleaseDate` | 是 | `str` | 最近解禁日 |
| `RecentDZDate` | 是 | `str` | 最近定增日 |
| `ReportDate` | 是 | `str` | 最近财报公告日期 |
| `ZTDate_Recent` | 是 | `str` | 近 2 年最近涨停板日期 |
| `DTDate_Recent` | 是 | `str` | 近 2 年最近跌停板日期 |
| `TopDate_Recent` | 是 | `str` | 近 2 年最近龙虎榜日期 |
| `StopJYDate_Recent` | 是 | `str` | 最近停牌日期 |

### 提示

涨停跌停的判断：用 `get_more_info` 取 `FCAmo` 来判断涨停跌停，大于 0 是涨停，小于 0 是跌停。

## 接口使用

```python
from tqcenter import tq

tq.initialize(__file__)

more_info = tq.get_more_info(stock_code='688318.SH', field_list=[])
print(more_info)
```
