# 获取股票所属板块 get_relation

获取指定股票所属板块信息。

## 函数签名

```python
def get_relation(stock_code: str = '')
```

## 输入参数

| 参数 | 是否必选 | 参数类型 | 参数说明 |
|------|----------|----------|----------|
| `stock_code` | 是 | `str` | 股票代码 |

## 返回数据

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `BlockCode` | 是 | `str` | 板块代码（没有板块代码的板块返回 `"0"`） |
| `BlockName` | 是 | `str` | 板块名称 |
| `BlockType` | 是 | `str` | 板块类型 |
| `GPNume` | 是 | `str` | 成份股数量 |

## 接口使用

```python
from tqcenter import tq
from tqcenter import tqconst

tq.initialize(__file__)

gp_block_res = tq.get_relation(stock_code='688318.SH')
print(gp_block_res)
```
