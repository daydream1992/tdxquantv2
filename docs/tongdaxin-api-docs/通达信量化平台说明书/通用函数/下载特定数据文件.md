# 下载特定数据文件 download_file

下载 10 大股东数据文件、ETF 申赎清单文件、最近舆情信息文件、股票综合信息文件。

## 函数签名

```python
download_file(stock_code: str = '',
              down_time: str = '',
              down_type: int = 1)
```

## 输入参数

| 参数 | 是否必选 | 参数类型 | 参数说明 |
|------|----------|----------|----------|
| `stock_code` | 是 | `str` | 证券代码 |
| `down_time` | 是 | `str` | 指定日期 |
| `down_type` | 是 | `int` | 指定下载类型 |

## down_type 说明

| 值 | 说明 |
|----|------|
| `1` | 下载 10 大股东数据文件，`down_time` 为指定日期，下载的文件中含指定日期所在年度的所有 10 大股东数据和流通股东数据 |
| `2` | 下载 ETF 申赎清单文件，`down_time` 为指定日期 |
| `3` | 下载最近舆情信息文件，其余两项无效 |
| `4` | 下载股票综合信息文件，其余两项无效 |

> 下载的文件保存在 `.\PYPlugins\data` 文件夹。

## 接口使用

```python
from tqcenter import tq

tq.initialize(__file__)

# 下载10大股东数据
down_ptr_10 = tq.download_file(stock_code='688318.SH', down_time='20241231', down_type=1)
print(down_ptr_10)

# 下载ETF申赎数据
down_ptr_etf = tq.download_file(stock_code='159109.SH', down_time='20260227', down_type=2)
print(down_ptr_etf)
```

## 数据样本

下载 10 大股东数据：

```json

   "Msg": "下载十大股东数据[2025]成功。",
   "run_id": "1"
}
```

```json
{
   "ErrorId": "0",
   "Msg": "下载ETF申述清单[20250101]成功。",
   "run_id": "1"
}
```
