# 检索证券信息 get_match_stkinfo

检索证券信息。

## 函数签名

```python
def get_match_stkinfo(key_word: str = '')
```

## 输入参数

| 参数 | 是否必选 | 参数类型 | 参数说明 |
|------|----------|----------|----------|
| `key_word` | 是 | `str` | 关键词 |

## 输出数据

| 名称 | 类型 | 说明 |
|------|------|------|
| `Code` | `str` | 证券代码 |
| `Name` | `str` | 证券名称 |

## 接口使用

```python
from tqcenter import tq

tq.initialize(__file__)

match_stkinfo = tq.get_match_stkinfo(key_word='通达信')
print(match_stkinfo)
```
