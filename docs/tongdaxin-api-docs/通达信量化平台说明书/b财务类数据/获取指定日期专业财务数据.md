# 获取指定日期专业财务数据 get_financial_data_by_date

根据股票，获取指定日期的专业财务数据。与基础财务数据不同，需要先在客户端中下载专业财务数据。

## 函数签名

```python
get_financial_data_by_date(stock_list: List[str] = [],
                           field_list: List[str] = [],
                           year: int = 0,
                           mmdd: int = 0) -> Dict
```

## 输入参数

| 参数 | 是否必选 | 参数类型 | 参数说明 |
|------|----------|----------|----------|
| `stock_list` | 是 | `List[str]` | 证券代码列表 |
| `field_list` | 是 | `List[str]` | 字段筛选，不能为空（如 `FN193`） |
| `year` | 是 | `int` | 指定年份 |
| `mmdd` | 是 | `int` | 指定月日 |

### 参数说明

`year` 和 `mmdd` 的组合规则：

| year | mmdd | 说明 |
|------|------|------|
| `0` | `0` | 取最新的财报 |
| `0` | `< 300` | 最近一期向前推 mmdd 期的数据 |
| `0` | `331/630/930/1231` | 最近一期的对应季报数据 |
| 数字 | `0` | 最近一期向前推 year 年的同期数据 |

> 季报分界点为：0331、0630、0930、1231。
> 需要先在客户端中下载财务数据包。

## 输出数据

同 `get_financial_data` 一样，返回 `Dict`，key 为股票代码，value 为字段名到值的映射。

## 接口使用

```python
from tqcenter import tq

tq.initialize(__file__)

fd = tq.get_financial_data_by_date(
    stock_list=['688318.SH'],
    field_list=['Fn193', 'Fn194', 'Fn195', 'Fn196', 'Fn197'],
    year=0,
    mmdd=0)
print(fd)
```
