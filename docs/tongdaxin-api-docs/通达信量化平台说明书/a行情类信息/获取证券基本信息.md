# 获取证券基本信息 get_stock_info

根据股票，获取股票基础的财务数据。

## 函数签名

```python
get_stock_info(cls,
               stock_code: str,
               field_list: List = []) -> Dict
```

## 输入参数

| 参数 | 是否必选 | 参数类型 | 参数说明 |
|------|----------|----------|----------|
| `stock_code` | 是 | `str` | 证券代码 |
| `field_list` | 是 | `List[str]` | 字段筛选，不能为空 |

## 返回数据

### 基本信息

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `Name` | 是 | `str` | 证券名称 |
| `Unit` | 是 | `str` | 交易单位 |
| `VolBase` | 是 | `str` | 量比的基量 |
| `MinPrice` | 是 | `str` | 最小价格变动 |
| `XsFlag` | 是 | `str` | 价格小数位数 |
| `Fz[8]` | 是 | `List[str]` | 开收市时间（4 段） |
| `DelayMin` | 是 | `str` | 延时分钟数 |
| `QHVolBaseRate` | 是 | `str` | 期货期权的每手乘数 |
| `HKVolBaseRate` | 是 | `str` | 港股/日股/新加坡股每手股数 |

### 分类标识

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `BelongHS300` | 是 | `str` | 是否属于沪深 300 |
| `BelongHasKQZ` | 是 | `str` | 是否含可转债 |
| `BelongRZRQ` | 是 | `str` | 是否是融资融券标的 |
| `BelongHSGT` | 是 | `str` | 是否属于沪深股通 |
| `IsHKGP` | 是 | `str` | 是否是港股 |
| `IsQH` | 是 | `str` | 是否是期货 |
| `IsQQ` | 是 | `str` | 是否是期权 |
| `IsSTGP` | 是 | `str` | 是否是 ST 股票 |
| `IsQuitGP` | 是 | `str` | 是否是退市整理板股票 |
| `TodayDRFlag` | 是 | `str` | 当天是否有除权除息（沪深京） |
| `HSStockKind` | 是 | `str` | 沪深京品种类型（见下表） |

**HSStockKind 品种类型：**

| 值 | 说明 |
|----|------|
| 0 | 指数 |
| 1 | A 股主板 |
| 2 | 北证 A 股 |
| 3 | 创业板 |
| 4 | 科创板 |
| 5 | B 股 |
| 6 | 债券 |
| 7 | 基金 |
| 8 | 权证 |
| 9 | 其它 |
| 10 | 非沪深京品种 |

### 股本与资产

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `ActiveCapital` | 是 | `str` | 流通股本（万股） |
| `J_zgb` | 是 | `str` | 总股本（万股） |
| `J_bg` | 是 | `str` | B 股（万股） |
| `J_hg` | 是 | `str` | H 股（万股） |
| `J_zzc` | 是 | `str` | 总资产（万元） |
| `J_ldzc` | 是 | `str` | 流动资产（万元） |
| `J_gdzc` | 是 | `str` | 固定资产（万元） |
| `J_wxzc` | 是 | `str` | 无形资产（万元） |
| `J_ldfz` | 是 | `str` | 流动负债（万元） |
| `J_cqfz` | 是 | `str` | 少数股东权益（万元） |
| `J_zbgjj` | 是 | `str` | 资本公积金（万元） |
| `J_jzc` | 是 | `str` | 股东权益/净资产（万元） |

### 收入与利润

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `J_yysy` | 是 | `str` | 营业收入（万元） |
| `J_yycb` | 是 | `str` | 营业成本（万元） |
| `J_yszk` | 是 | `str` | 应收账款（万元） |
| `J_yyly` | 是 | `str` | 营业利润（万元） |
| `J_tzsy` | 是 | `str` | 投资收益（万元） |
| `J_jyxjl` | 是 | `str` | 经营现金净流量（万元） |
| `J_zxjl` | 是 | `str` | 总现金净流量（万元） |
| `J_ch` | 是 | `str` | 存货（万元） |
| `J_lyze` | 是 | `str` | 利润总额（万元） |
| `J_shly` | 是 | `str` | 税后利润（万元） |
| `J_jly` | 是 | `str` | 净利润（万元） |
| `J_wfply` | 是 | `str` | 未分配利益（万元） |

### 每股指标

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `J_jyl` | 是 | `str` | 净资产收益率 |
| `J_mgwfp` | 是 | `str` | 每股未分配 |
| `J_mgsy` | 是 | `str` | 每股收益（折算为全年） |
| `J_mgsy2` | 是 | `str` | 季报每股收益（财报中提供的每股收益） |
| `J_mggjj` | 是 | `str` | 每股公积金 |
| `J_mgjzc` | 是 | `str` | 每股净资产 |
| `J_mgjzc2` | 是 | `str` | 季报每股净资产（财报中提供的每股收益） |
| `J_gdqyb` | 是 | `str` | 股东权益比 |
| `J_gdrs` | 是 | `str` | 股东人数 |
| `J_HalfYearFlag` | 是 | `str` | 报告期月份（3, 6, 9, 12） |

### 其他信息

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `J_start` | 是 | `str` | 上市日期 |
| `tdx_dycode` | 是 | `str` | 通达信地域代码 |
| `tdx_dyname` | 是 | `str` | 通达信地域 |
| `rs_hycode_sim` | 是 | `str` | 通达信行业代码 |
| `rs_hyname` | 是 | `str` | 通达信行业 |
| `blockzscode` | 是 | `str` | 所属的行业板块指数代码 |
| `underly_setcode` | 是 | `str` | 标的市场代码（如当前 ETF 跟踪的指数市场） |
| `underly_code` | 是 | `str` | 标的代码（如当前 ETF 跟踪的指数代码） |

## 接口使用

```python
from tqcenter import tq

tq.initialize(__file__)

fdc = tq.get_stock_info(stock_code='688318.SH', field_list=[])
print(fdc)
```
