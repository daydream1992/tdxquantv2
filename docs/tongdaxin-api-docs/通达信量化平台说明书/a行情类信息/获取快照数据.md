# 获取快照数据 get_market_snapshot

根据股票，获取最新行情数据。

## 函数签名

```python
def get_market_snapshot(stock_code: str,
                        field_list: List = []) -> Dict
```

## 输入参数

| 参数 | 是否必选 | 参数类型 | 参数说明 |
|------|----------|----------|----------|
| `stock_code` | 是 | `str` | 证券代码 |
| `field_list` | 否 | `List[str]` | 字段筛选，传空则返回全部 |

## 返回数据

| 字段 | 默认返回 | 数据类型 | 说明 |
|------|----------|----------|------|
| `ItemNum` | 是 | `str` | 快照笔数 |
| `LastClose` | 是 | `str` | 前收盘价 |
| `Open` | 是 | `str` | 开盘价 |
| `Max` | 是 | `str` | 最高价 |
| `Min` | 是 | `str` | 最低价 |
| `Now` | 是 | `str` | 现价 |
| `Volume` | 是 | `str` | 总手 |
| `NowVol` | 是 | `str` | 现手 |
| `Amount` | 是 | `str` | 总成交金额 |
| `Inside` | 是 | `str` | 内盘（板块指数时为跌停家数） |
| `Outside` | 是 | `str` | 外盘（板块指数时为涨停家数） |
| `TickDiff` | 是 | `str` | 笔涨跌 |
| `InOutFlag` | 是 | `str` | 内外盘标志：0 Buy 1 Sell 2 Unknown |
| `Jjjz` | 是 | `str` | 基金净值 |
| `Buyp` | 是 | `List[str]` | 五个买价 |
| `Buyv` | 是 | `List[str]` | 对应的五个买盘量 |
| `Sellp` | 是 | `List[str]` | 五个卖价 |
| `Sellv` | 是 | `List[str]` | 对应的五个卖盘量 |
| `UpHome` | 是 | `str` | 上涨家数（对于指数有效） |
| `DownHome` | 是 | `str` | 下跌家数（对于指数有效） |
| `Before5MinNow` | 是 | `str` | 5 分钟前价格 |
| `Average` | 是 | `str` | 均价 |
| `XsFlag` | 是 | `str` | 小数位数 |
| `Zangsu` | 是 | `str` | 涨速 |
| `ZAFPre3` | 是 | `str` | 3 日涨幅 |

## 接口使用

```python
from tqcenter import tq

tq.initialize(__file__)

market_snapshot = tq.get_market_snapshot(stock_code='688260.SH', field_list=[])
print(market_snapshot)
```
