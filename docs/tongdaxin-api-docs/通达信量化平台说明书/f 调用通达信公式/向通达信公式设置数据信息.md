# 向通达信公式设置数据信息 formula_set_data_info

在调用公式前须先设置公式参数，此接口与 `formula_set_data` 作用一样，会互相覆盖。

## 函数签名

```python
def formula_set_data_info(stock_code: str = '',
                          stock_period: str = '1d',
                          start_time: str = '',
                          end_time: str = '',
                          count: int = -1,
                          dividend_type: int = 0)
```

## 输入参数

| 参数 | 是否必选 | 参数类型 | 参数说明 |
|------|----------|----------|----------|
| `stock_code` | 是 | `str` | 股票代码 |
| `stock_period` | 是 | `str` | K 线周期 |
| `start_time` | 是 | `str` | 起始时间 |
| `end_time` | 是 | `str` | 结束时间 |
| `count` | 是 | `int` | 截取 K 线数量 |
| `dividend_type` | 是 | `int` | 复权类型 |

## 参数说明

### 复权类型

| 值 | 说明 |
|----|------|
| `0` | 不复权 |
| `1` | 前复权 |
| `2` | 后复权 |

### count 参数

| 值 | 行为 |
|------|------|
| `> 0` | 截取最新交易日开始往前的 n 条 K 线，此时 `start_time` 和 `end_time` 失效 |
| `-1` | 获取所有数据 |
| `-2` | 使用无序列数据 |
| `0` | `start_time` 和 `end_time` 生效，指定对应时间段内的 K 线 |

> `count` 最大值为 24000。

### 注意事项

- 需要先在客户端下载对应的盘后数据。
- 设置的数据在断开连接前一直生效，后设置的数据会覆盖前面设置的数据。

## 接口使用

```python
from tqcenter import tq

tq.initialize(__file__)

formula_set_res = tq.formula_set_data_info(
    stock_code='688318.SH',
    stock_period='1d',
    count=100,
    dividend_type=1)
print(formula_set_res)
```
